## v0.1.61

### Changed
- Servers tab now has two dedicated top actions (`Add Server`, `Bulk Create Servers`) and each opens its own collapsible section.
- Server forms and inline edit forms are centered for cleaner editing flow.
- Removed `Asset Tag` and `Serial` from server create workflow.

### Added
- Datacenter edit now supports `Rack Count` and `Rack Units` and can auto-create missing racks.
- Rack edit now includes `Row`, `Rack`, and `Notes` fields.

### Fixed
- Fixed Safari/WebKit selector escaping issues causing `SyntaxError: Invalid escape in identifier` in server/rack/switch JS.
- Switch edit rack list is now filtered by selected datacenter.
