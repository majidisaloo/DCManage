## Changed
- Rack display is now front-only for clearer and faster unit operations.
- Clicking a unit in the rack view now selects that U automatically in the unit configuration form.
- Datacenter and switch action buttons were restyled to modern grouped controls.
- `Networks` tab/action were removed from admin navigation.

## Added
- Inline rack rename and total-U update controls in rack cards.
- Switch management flow now matches Datacenter pattern with collapsible add form.
- Switch vendor options now include `Cisco`, `Nexus`, and `MikroTik`.
- SNMP controls on switch forms: IP, version, port, community.
- Per-switch SNMP connectivity test with status badge (`UP`/`DOWN`).
- Per-switch ports and VLAN management: list, add/update, delete.
- New table: `mod_dcmanage_switch_ports` for switch interface/VLAN inventory.
