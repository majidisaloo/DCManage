# v0.1.75

## Fixed
- Improved VLAN detection/parsing (`Vl*` formats and numeric normalization).
- Fixed Monitoring and Servers black-screen behavior caused by forced static modal rendering.
- Removed synchronous PRTG counter pulls from server details render flow to reduce timeout/blank-page risk.
- Fixed monitoring add/view markup imbalance that could break layout rendering.

## Changed
- Datacenter, Switch, and Traffic Package create flows now use modals instead of inline collapses.
- Monitoring edit now opens in modal flow, aligned with add/view behavior.
- Services/Group action now explicitly uses `Load Products`.
- Traffic dashboard sort labels are now human-friendly.
- Improved responsive layout and spacing for tables, buttons, and modal form sections.

## Added
- Added `scope_load_products` translation key in English/Persian.
