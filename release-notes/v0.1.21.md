### Added
- Added SNMP compatibility layer for discovery with support for:
  - `snmp2_real_walk`
  - `snmprealwalk`
  - fallback `snmp2_walk` / `snmpwalk`
- Added SNMP get compatibility fallback (`snmp2_get` -> `snmpget`) for connectivity checks.

### Fixed
- Fixed auto-discovery failures on hosts where `snmp2_real_walk` is unavailable.
- Improved switch port discovery resilience across different PHP SNMP function sets.
