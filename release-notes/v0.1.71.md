# v0.1.71

## Changed
- `Services / Group` now uses CSV-based `GID` and `PID` filters.
- Combined filter behavior supports union loading (`GID + PID`).
- Default scope page no longer loads all products without explicit filters.
- Group default quota editors now render for each selected GID.

## Fixed
- CSV parser now supports Persian/Arabic digits for GID/PID values.
