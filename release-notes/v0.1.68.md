## Added
- Implemented full `Services / Group` admin tab for WHMCS scope defaults.
- Added product/group loading (PID/GID) from WHMCS with filtering.
- Added per-product traffic defaults: Download GB, Upload GB, Total GB.
- Added unlimited toggles for each limit with auto-disable UX.
- Added group-level default form for applying scope baselines by GID.
- Added Datacenter traffic calculation mode field (`Download` / `Upload` / `Total`) in create/edit forms.

## Changed
- Usage engine now enforces traffic based on Datacenter calculation mode.
- Usage engine now resolves default quota using scope hierarchy (PID first, then GID fallback).
- Renamed tab label from `WHMCS Scope` to `Services / Group` (EN/FA updated).

## Fixed
- Datacenter list query updated to safely include traffic calculation mode in grouped select.
- Improved scope row checkbox/input spacing and action alignment for cleaner editing.
