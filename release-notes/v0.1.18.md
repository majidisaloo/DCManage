### Changed
- Switches tab UI was modernized with cleaner action buttons, clearer spacing, and a compact card-style ports panel.
- Port add/update form now uses explicit labels for Interface, VLAN, Admin, and Oper to prevent RTL/LTR input confusion.

### Added
- Added `Discover Ports` action per switch to auto-import interface list and link status from SNMP walk.
- Added colored status pills for SNMP switch state and per-port admin/oper state.

### Fixed
- Dashboard `Ports` counter now counts both `switch_ports` and `server_ports`.
- Interface normalization improved so values like `Ethernet 1/1` are saved as `Ethernet1/1`.
- UP/DOWN state no longer renders as neutral gray; connected states are now explicitly green/red.
